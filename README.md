### 👨‍💻 About Me

C++ developer specializing in medical image processing and visualization.
I work with ITK, VTK, and Qt/QML, developing tools for 3D segmentation, volume rendering, and DICOM data analysis.

🔬 Key areas:

- 3D segmentation (Level Set, Fast Marching, Sparse Field)

- Compute optimization and memory consumption reduction

- Working with DICOM, MHA, and medical tomography data

- Integration of ITK algorithms into VTK/Qt applications

- Curved MPR, panoramic reconstructions, volume cropping

- Marching Cubes and 3D visualization


Focus on the performance, architecture of medical imaging systems, and deep integration of image processing algorithms into applications.
